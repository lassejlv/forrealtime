---
title: Redis Adapters
description: Use ioredis, Bun.redis, or write your own Redis adapter.
---

forrealtime uses an adapter pattern so you can bring your own Redis client. Two official adapters are included, and you can write your own.

- [ioredis](/docs/adapters/ioredis) — Use the popular ioredis client
- [Bun Redis](/docs/adapters/bun) — Use Bun's built-in Redis client
- [Custom Adapter](/docs/adapters/custom) — Implement the RedisAdapter interface for any Redis client
